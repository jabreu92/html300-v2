Using ES6+ Methods in DOM Interactions

Map Function, we are going to add the favorite number to the array [0, 1, 2, 3, 5, 8]

Filter Function, we are going to get elements that are greater than the favorite number in the [0, 1, 2, 3, 5, 8] array

The Every Function will check if every single element in the array [1, 2, 3, 4, 5, 6] is greater than the favorite number

The Some Function will check if any element in the array [0, 1, 2, 3, 5, 8] is greater than the favorite number
